**限制**：4000 個字符以內

```
LifeLab - 您的專屬生命藍圖

LifeLab 是一款由 AI 驅動的個人化職涯規劃應用，透過深度分析您的興趣、優勢、價值觀與個人背景，為您量身打造專屬的生命藍圖與職涯發展方向。

核心功能

深度自我探索
透過科學化的問卷與 AI 分析，深入了解您的興趣、優勢、價值觀與個人特質，建立完整的自我認知。

AI 生成生命藍圖
基於您的個人資料，AI 為您生成 3 個量身打造的職涯方向，每個方向都包含詳細的市場可行性分析與發展建議。

深化探索
持續優化您的生命藍圖，透過更深入的探索與分析，找到最適合您的職涯路徑。

行動計劃制定
將抽象的職涯目標轉化為具體的行動計劃，包含短期、中期與長期目標，以及關鍵里程碑。

任務管理
每日任務提醒與追蹤，幫助您逐步實現職涯目標，讓夢想不再遙遠。

個人檔案管理
完整記錄您的探索歷程與生命藍圖版本，隨時回顧與調整。

為什麼選擇 LifeLab？

個人化分析：每個生命藍圖都是為您量身打造，絕非通用模板
AI 驅動：運用先進的 AI 技術，提供專業且深入的職涯建議
科學方法：基於心理學與職涯發展理論，確保建議的專業性與可靠性
持續優化：支援多版本生命藍圖，讓您隨著成長持續調整方向
行動導向：不只是分析，更提供具體的行動計劃與任務管理

適合對象

正在尋找職涯方向的學生與新鮮人
考慮轉職或轉換跑道的職場人士
希望重新規劃人生方向的任何人
想要深入了解自己的每一個人

使用流程

1. 完成個人資料問卷（基本資訊、興趣、優勢、價值觀）
2. AI 分析並生成初版生命藍圖
3. 深化探索，優化生命藍圖
4. 制定行動計劃，設定目標與里程碑
5. 每日執行任務，逐步實現夢想

隱私與安全

我們重視您的隱私，所有個人資料均經過加密處理，僅用於生成您的專屬生命藍圖。您可以隨時查看、編輯或刪除您的資料。

支援與反饋

如有任何問題或建議，歡迎透過應用內的意見反饋功能與我們聯繫。

開始您的生命藍圖之旅，探索無限可能！
```

## 🌐 支援 URL（Support URL）

**必需**：必須是可訪問的網頁

### 選項 1: 使用 GitHub Pages（免費）

如果您有 GitHub 帳號：
1. 創建一個簡單的 HTML 頁面
2. 使用 GitHub Pages 託管
3. URL 格式：`https://yourusername.github.io/lifelab-support`

### 選項 2: 使用 Supabase（推薦）

如果您使用 Supabase：
1. 創建一個簡單的支持頁面
2. 託管在 Supabase Storage 或 Edge Functions
3. URL 格式：`https://yourproject.supabase.co/support`

### 選項 3: 使用簡單的網頁服務

可以使用任何免費網頁託管服務：
- Netlify
- Vercel
- Firebase Hosting

