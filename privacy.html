LifeLab 隱私政策

最後更新日期：2024年2月21日

1. 資料收集

我們收集以下資料以提供 LifeLab 服務：

  • 個人基本資料（可選）：地區、年齡、稱謂、職業、年薪、家庭狀況、教育程度
  • 興趣與優勢：您選擇的興趣關鍵字、優勢問卷回答、優勢關鍵詞
  • 價值觀：核心價值觀排序與問卷回答
  • 深化探索資料（可選）：心流日記、價值觀反思、資源盤點、後天強項、可行性評估
  • 生命藍圖資料：AI 生成的職涯方向、行動計劃、任務資料
  • 認證資料：電子郵件地址（用於帳號登入和認證）
  • 應用使用情況：功能使用記錄、錯誤日誌（用於改進服務）

重要說明：

  • 我們不會收集您的姓名、電話號碼或設備識別碼
  • 基本資料和深化探索資料為可選填寫，您可以選擇不提供
  • 我們僅在您明確同意後才會將資料發送給第三方 AI 服務

2. 資料使用

我們使用收集的資料用於以下目的：

  • 生成您的專屬生命藍圖與職涯建議（僅在您明確同意後）
  • 提供個人化的行動計劃與任務建議
  • 改進應用功能與使用者體驗
  • 提供客戶支援服務
  • 確保應用正常運作與安全性
  • 處理訂閱與支付（透過 Apple App Store）

AI 服務使用：

  • 我們僅在您點擊「開啟我的理想人生」按鈕（明確同意）後，才會將您提供的資料發送給第三方 AI 服務（AIML API / Anthropic Claude）
  • 發送給 AI 服務的資料不包含任何個人識別資訊（如姓名、電子郵件、電話號碼）
  • AI 服務提供商不會將您的資料用於訓練 AI 模型

3. 資料儲存與安全

我們採用以下措施保護您的資料安全：

  • 加密傳輸：所有資料傳輸均使用 HTTPS 加密
  • 安全儲存：資料儲存在 Supabase 安全伺服器上，採用業界標準加密技術
  • 本地快取：資料會在您的設備上進行本地快取，確保快速載入和離線使用
  • 存取控制：只有您本人可以存取您的個人資料

我們使用 Supabase 作為資料庫服務提供商，他們遵循嚴格的資料安全標準。

4. 資料分享

我們重視您的隱私，不會將您的個人資料出售或分享給第三方，除非：

  • 法律要求或法院命令
  • 保護我們的權利、財產或安全
  • 經您明確同意
  • 與服務提供商分享（如 Supabase），但僅限於提供服務所需

我們不會將您的資料用於廣告或行銷目的。

5. 您的權利

您對您的個人資料擁有以下權利：

  • 查看權：隨時查看您的個人資料
  • 修改權：修改或更新您的資料
  • 刪除權：刪除您的帳號和所有資料
  • 導出權：導出您的所有資料
  • 撤回同意：隨時撤回資料處理同意

您可以透過應用內的「設定」→「導出所有資料」或「清除所有資料」功能行使這些權利。

6. 第三方服務

我們使用以下第三方服務：

  • Supabase：用於資料儲存與用戶認證服務。Supabase 遵循嚴格的資料安全標準，並採用 Row Level Security (RLS) 確保只有您本人可以存取您的資料。
  • AI 服務（AIML API / Anthropic Claude Sonnet 4.5）：用於生成個人化生命藍圖與職涯建議。我們僅在您明確同意（點擊「開啟我的理想人生」按鈕）後，才會將您提供的資料發送給 AI 服務。我們不會發送任何個人識別資訊（如姓名、電子郵件、電話號碼）給 AI 服務。所有資料傳輸均使用 HTTPS 加密。
  • Apple Sign In：用於 Apple 帳號登入。Apple 的隱私政策適用於此服務。
  • Apple App Store：用於應用分發與訂閱支付處理。Apple 的隱私政策適用於此服務。

重要說明：

  • AI 服務提供商不會將您的資料用於訓練 AI 模型（根據 AIML API 的隱私政策）
  • 我們不會將您的資料出售或用於廣告目的
  • 所有第三方服務提供商都有各自的隱私政策，我們建議您查看相關政策

7. 資料保留

我們會保留您的資料，直到您要求刪除或帳號被刪除。刪除帳號後，我們會在合理時間內刪除您的所有個人資料。

8. 兒童隱私

LifeLab 不適合 13 歲以下兒童使用。我們不會故意收集 13 歲以下兒童的個人資料。如果我們發現收集了兒童資料，我們會立即刪除。

9. 國際資料傳輸

您的資料可能會被傳輸到並儲存在您所在國家/地區以外的伺服器上。我們會確保這些傳輸符合適用的資料保護法律。

10. 政策變更

我們可能會不時更新本隱私政策。重大變更將透過應用通知您，並在政策頂部標註更新日期。繼續使用應用即表示您接受更新後的政策。

11. 聯絡我們

如有任何關於本隱私政策的問題、意見或請求，請透過以下方式聯繫我們：

  • 應用內「設定」→「意見反饋」功能
  • 電子郵件：privacy@lifelab.app

我們會盡快回覆您的問題。

本隱私政策適用於 LifeLab 應用及其所有功能與服務。
