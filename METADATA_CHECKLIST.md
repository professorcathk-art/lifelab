# App Store Connect Metadata 完整检查清单

## Apple Guideline 2.3.2 - 需要检查的所有 Metadata 字段

### ✅ 必须修改的字段

#### 1. **应用描述（Description）** ⭐ 最重要
- **位置**：App Store Connect → App 信息 → 描述
- **必须包含**：
  - 明确说明需要订阅
  - 列出需要订阅的功能
  - 订阅价格信息
- **状态**：✅ 需要修改

#### 2. **副标题（Subtitle）**（如果有）
- **位置**：App Store Connect → App 信息 → 副标题
- **检查**：如果副标题提到功能，需要明确说明需要订阅
- **示例**：
  - ❌ "AI 驅動的人生規劃工具"
  - ✅ "需要訂閱的 AI 人生規劃工具"
- **状态**：⚠️ 需要检查

#### 3. **促销文本（Promotional Text）**（如果有）
- **位置**：App Store Connect → App 信息 → 促销文本
- **检查**：如果提到功能，需要明确说明需要订阅
- **状态**：⚠️ 需要检查

### ⚠️ 可能需要修改的字段

#### 4. **截图（Screenshots）**
- **位置**：App Store Connect → 版本 → 截图
- **检查**：
  - 截图中的文字是否提到功能但没有说明需要订阅？
  - 截图是否暗示功能是免费的？
- **如果截图有问题**：
  - 在截图上添加文字说明"需要订阅"
  - 或者重新制作截图
- **状态**：⚠️ 需要检查

#### 5. **应用预览视频（App Preview）**（如果有）
- **位置**：App Store Connect → 版本 → 应用预览
- **检查**：
  - 视频中是否提到功能但没有说明需要订阅？
  - 视频是否暗示功能是免费的？
- **如果视频有问题**：
  - 在视频中添加文字说明"需要订阅"
  - 或者重新制作视频
- **状态**：⚠️ 需要检查（如果有）

#### 6. **关键词（Keywords）**（通常不需要改）
- **位置**：App Store Connect → App 信息 → 关键词
- **检查**：关键词通常不需要修改，除非关键词本身暗示免费
- **状态**：✅ 通常不需要修改

## 📋 完整检查步骤

### Step 1: 检查所有 Metadata 字段

登录 App Store Connect，检查以下所有字段：

1. ✅ **描述（Description）** - 必须修改
2. ⚠️ **副标题（Subtitle）** - 检查是否需要修改
3. ⚠️ **促销文本（Promotional Text）** - 检查是否需要修改
4. ⚠️ **截图（Screenshots）** - 检查是否需要修改
5. ⚠️ **应用预览（App Preview）** - 检查是否需要修改（如果有）
6. ✅ **关键词（Keywords）** - 通常不需要修改

### Step 2: 修改原则

**所有提到功能的地方，都必须明确说明需要订阅！**

#### 描述（Description）- 必须修改
```
✅ 必须包含：
- 明确说明需要订阅
- 列出需要订阅的功能
- 订阅价格信息
- 自动续订说明
```

#### 副标题（Subtitle）- 如果存在，需要检查
```
❌ 错误示例：
"AI 驅動的人生規劃工具"

✅ 正确示例：
"需要訂閱的 AI 人生規劃工具"
或
"訂閱制 AI 人生規劃應用"
```

#### 截图（Screenshots）- 如果提到功能，需要检查
```
如果截图中有文字提到功能：
❌ "獲得個人化生命藍圖"
✅ "訂閱後獲得個人化生命藍圖"

或者：
✅ 在截图上添加"需要訂閱"标签
```

### Step 3: 修改优先级

1. **高优先级（必须修改）**：
   - ✅ 应用描述（Description）

2. **中优先级（建议检查）**：
   - ⚠️ 副标题（Subtitle）
   - ⚠️ 促销文本（Promotional Text）

3. **低优先级（如果有问题才改）**：
   - ⚠️ 截图（Screenshots）
   - ⚠️ 应用预览（App Preview）

## 🎯 快速检查清单

在 App Store Connect 中，逐一检查：

- [ ] **描述（Description）**：是否明确说明需要订阅？
- [ ] **副标题（Subtitle）**：如果存在，是否提到需要订阅？
- [ ] **促销文本（Promotional Text）**：如果存在，是否提到需要订阅？
- [ ] **截图（Screenshots）**：是否暗示功能是免费的？
- [ ] **应用预览（App Preview）**：是否暗示功能是免费的？

## 💡 建议

### 最安全的做法：

1. **先修改描述（Description）** - 这是最重要的
2. **检查副标题和促销文本** - 如果有，确保提到订阅
3. **检查截图** - 如果截图中有文字提到功能，确保说明需要订阅
4. **提交修改** - 先提交这些修改，看 Apple 是否还有反馈

### 如果 Apple 还有反馈：

如果修改描述后，Apple 还提到其他 metadata 字段有问题，再修改那些字段。

## 📝 总结

**回答你的问题：**

**主要修改描述（Description）就可以，但建议也检查一下：**
- ✅ **描述（Description）** - 必须修改
- ⚠️ **副标题（Subtitle）** - 如果有，建议检查
- ⚠️ **促销文本（Promotional Text）** - 如果有，建议检查
- ⚠️ **截图（Screenshots）** - 如果提到功能，建议检查

**最安全的做法：先修改描述，然后检查其他字段，确保所有提到功能的地方都明确说明需要订阅。**
