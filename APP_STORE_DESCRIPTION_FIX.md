# Apple Guideline 2.3.2 - App Description Fix

## 问题说明

**Apple 反馈**：应用的元数据（App Store Connect 中的应用描述）提到了付费内容或功能，但没有明确标识这些内容需要额外购买。具体来说，应用需要用户购买订阅才能访问，但应用描述没有告知用户需要购买才能访问这些内容。

## ✅ 解决方案

**你只需要在 App Store Connect 中修改应用描述，不需要修改代码！**

### 需要修改的地方

1. **App Store Connect** → **你的应用** → **App 信息** → **描述**
2. **App Store Connect** → **你的应用** → **App 信息** → **副标题**（如果有）
3. **App Store Connect** → **你的应用** → **App 信息** → **关键词**（如果有提到付费功能）

### 修改原则

✅ **必须明确说明**：
- 应用需要订阅才能使用
- 哪些功能需要订阅
- 订阅价格和周期

❌ **不要**：
- 只说功能，不提需要付费
- 暗示功能是免费的
- 隐藏订阅要求

## 📝 修改建议

### 当前描述可能的问题示例（错误）

```
❌ "LifeLab 幫助你通過 AI 分析找到人生方向，提供個人化生命藍圖和行動計劃。"
   - 问题：没有提到需要订阅

❌ "解鎖專屬 AI 深度分析，從自我洞察到落地執行。"
   - 问题：只说"解鎖"，但没有明确说明需要付费订阅
```

### 修改后的描述示例（正确）

#### 选项 1：在开头明确说明

```
✅ "LifeLab 是一個需要訂閱的應用，幫助你通過 AI 分析找到人生方向。

訂閱後，你可以獲得：
• 個人化生命藍圖生成
• AI 驅動的深度分析
• 專屬行動計劃和里程碑
• 進階深度探索功能

訂閱價格：月付 $17.99 / 季付 $29.99 / 年付 $89.99（節省 58%）
訂閱將自動續訂，可在 App Store 設置中管理。"
```

```
✅ "LifeLab 幫助你通過 AI 分析找到人生方向。
```

## 📋 修改步骤

### Step 1: 登录 App Store Connect

1. 访问 [appstoreconnect.apple.com](https://appstoreconnect.apple.com)
2. 登录你的开发者账户
3. 选择你的应用（LifeLab）

### Step 2: 编辑应用描述

1. 点击 **"App 信息"** 或 **"App Information"**
2. 找到 **"描述"** 或 **"Description"** 字段
3. 点击 **"编辑"** 或 **"Edit"**

### Step 3: 修改描述

1. **删除或修改**任何暗示功能是免费的内容
2. **添加**明确的订阅说明
3. **添加**订阅价格信息
4. **添加**自动续订说明

### Step 4: 检查其他字段

检查以下字段是否也需要修改：

- **副标题（Subtitle）**：如果有，确保提到订阅
- **关键词（Keywords）**：如果提到付费功能，确保明确
- **促销文本（Promotional Text）**：如果有，确保提到订阅

### Step 5: 保存并提交

1. 点击 **"保存"** 或 **"Save"**
2. 如果应用正在审核中，描述修改会自动更新
3. 如果应用已发布，描述修改会在下次更新时生效

## ✅ 检查清单

修改描述后，确保：

- [ ] 明确说明应用需要订阅
- [ ] 列出需要订阅的功能
- [ ] 包含订阅价格信息
- [ ] 说明自动续订机制
- [ ] 说明如何取消订阅
- [ ] 没有暗示功能是免费的
- [ ] 描述准确反映应用的实际功能

## 📝 给 Apple 的回复建议

提交修改后，可以在 App Review 回复中说明：

```
我们已经更新了应用描述，明确说明：

1. 应用需要订阅才能使用核心功能
2. 列出了所有需要订阅的功能
3. 包含了订阅价格和周期信息
4. 说明了自动续订机制和如何取消

应用描述现在完全符合 Guideline 2.3.2 的要求。
```

## 🌍 多语言支持

如果你有多个语言版本的应用描述，**所有语言版本都需要修改**：

- 简体中文
- 繁体中文
- English（如果有）
- 其他语言（如果有）

## ⚠️ 重要提示

1. **不需要修改代码**：这是纯元数据问题，只需要修改 App Store Connect 中的描述
2. **描述修改立即生效**：修改后，新用户会立即看到更新后的描述
3. **不影响现有用户**：描述修改不会影响已安装应用的用户
4. **可以多次修改**：如果 Apple 还有反馈，可以继续修改描述
