# LifeLab Website

This is the website for LifeLab app, containing support and privacy policy pages.

## Structure

- `index.html` - Main landing page
- `support.html` - Support and FAQ page
- `privacy.html` - Privacy policy page

## Deployment

This website is deployed on Vercel.

## URLs

- Support URL: `https://lifelab.vercel.app/support.html`
- Privacy Policy URL: `https://lifelab.vercel.app/privacy.html`
